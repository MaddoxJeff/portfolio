#INFO

Given a string containing only lowercase letters, 
find whether every letter that appears in the string appears the same number of times. 

example: 

balanced("xxxyyyzzz") => true
balanced("abccbaabccba") => true
balanced("xxxyyyzzzz") => false
balanced("abcdefghijklmnopqrstuvwxyz") => true
balanced("pqq") => false
balanced("fdedfdeffeddefeeeefddf") => false
balanced("www") => true
balanced("x") => true
balanced("") => true
